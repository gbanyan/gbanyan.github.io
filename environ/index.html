軟硬體環境

記載了目前打造的工作環境，定時更新

  • 2023.4 初稿
  • 2023.5 更新 Synergy
  • 2024.2 年度更新

筆電

  • 個人工作主力為 macOS，目前為一台最低階的 MacBook Air M1
  • 其僅能輸出一台外接螢幕，習慣在家以 Thunderbolt 連接 4K 螢幕
  • 以闔蓋模式或上下一大一小雙螢幕工作
  • Synergy, 融合 Linux + macOS 或 Windows + macOS 桌面

macOS 環境

  • 系統

    • yabai: Windows Tiling Manager, 自動分配並平鋪視窗
      • skhd 原 repo 貌似已不更新，目前搭配 raycast 自訂快速鍵使用
    • BetterDisplay: 付費軟體，比 MonitorControl 多了切換外接螢幕開關及其他實用功能，對常切換外接螢幕輸入源的環境很有幫助
      • MonitorControl: 調整內建螢幕及外接螢幕亮度，透過 DDC/CI, 支援熱鍵直接調整
    • SetApp: App Store 以外的訂閱制 App 來源，有不少好用的工具
    • Surge: 強大網路代理工具，拿來建立出門在外，並用公用網路與區域網路私有自架服務。
    • Microsoft Remote Desktop: 遠端桌面操控桌機
    • Vaultwarden: 自架密碼庫，並僅限內網存取，替換掉先前訂閱的 1Password。
    • Synergy: 建立 macOS 與其他作業系統的多重桌面

  • 效率

    • Raycast: 多功能快速啟動器，可搜尋、操控視窗、快速操控或搜尋特定應用程式內容、或執行自訂 Script
    • Logseq: 主力 GTD 以及資料筆記管理軟體，盡力養成每天記錄日誌的習慣
      • Obsidian: 定位是除 Logseq 外特定主題的知識筆記存放庫，不過目前少用。
    • Things: 待辦事項，可建立定期重複事項，有多重層級分類。
    • Mimestream: 桌面 Gmail Client, 簡潔快速，但是考慮取消訂閱
    • TexSnipper: 常駐在選單列的小巧 OCR 光學字元辨識工具，支援中英文

  • 文獻

    • PDF Expert: 比內建「預覽」好用的地方多了 Night, Sepia theme，以及垂直水平分割模式
    • Zotero: 跟 Firefox 同樣 Gecko 核心的文獻管理，簡單強大，可以直接在 Microsoft Word 內整理文獻，有 Dark Mode 外掛
    • MacGPT: 目前習慣使用的 OpenAI ChatGPT 桌面客戶端。
    • memos (自架服務): 收藏剪貼短篇筆記文章，可以快速與人分享。
    • MindNode: 心智圖繪製，簡單好用。
    • Typora: 簡單好用的 Markdown Editor
    • Apple Notes: 存放日常雜事快速筆記，蘋果裝置同步最無礙。
    • Mathpix: 最近在學習使用 Latex, 可以截圖自動辨識對應的 Latex 語法。
    • Microsoft Office and Apple iWork: 同時使用
      • 簡報目前習慣先開 Keynote 做個快速初稿，有比較才轉到 Powerpoint

  • 開發

    • Visual Studio Code: 裝一堆外掛，主力。
    • Sublime Text: 快速簡單好用，把他當成類 Windows 內建記事本來用了
    • iTerm2: 老牌終端機
    • Zsh + zimfw: 增強型 Shell, 可以減少繁複的指令切換
    • Forklift: 老牌雙欄分割檔案管理員，支援多種遠端傳輸協定
    • Hugo: 撰寫部落格以及部分網站的靜態網站框架
    • Proxmox VE：管理軟路由以及家中所有 Self-hosted 服務，偶爾部署資料爬蟲
    • Ollama: Local LLM 部署測試管理工具

  • 影音圖片

    • IINA: 包 mpv 的小巧播放器
    • Infuse: 可連接 NAS 並支援多重影音格式播放，定位是影音庫管理員。
    • PixelImator Pro: 簡潔但兼顧專業的修圖軟體

  • 其他

    • Noir: Safari 外掛，將所有網頁轉換至黑底白字，可以自訂白名單。
    • Arc Browser: 與 Safari 合併使用，在多分頁切換及群組管理很方便

    偶爾使用其他瀏覽器，單純區隔瀏覽紀錄與書籤用

  • 使用 Powershell 做 Linux 開機

Linux
