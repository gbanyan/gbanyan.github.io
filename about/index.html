關於本站

網站

此網站以 Hugo 架設，為個人部落格重新開機，舊的部落格放棄經營， 如果有空會搬遷一部分文章至此

未來此網站以職涯發展、知識學習筆記經營為重，偶爾會放上一些見聞心得 如同名字所述，既然人生走一遭，人間偶語，心境隨轉，船到橋頭自然直。

霍德爾之目

  • 北歐神話的神明，霍德爾（Höðr）
  • 奧丁與弗麗嘉之子，是光明之神巴德爾的孿生兄弟，掌管黑暗，生來即目盲。
  • 據說洛基將槲寄生樹枝交給目盲的霍德爾，誘騙他攻擊巴德爾，導致巴德爾的死亡。

網站歷程

  • 主機經歷多次搬遷，曾使用過 Digitealocean, Vultr, 以及 Google Cloud，最後落腳於 Github Pages
  • 最初使用 Wordpress 架站，後來改用 Ghost，體驗上最為滿意是 Ghost，不過最終改用 Hugo
  • 經營目的曾在學習網站技術、經營內容主題網站、單純個人筆記之間搖擺，最終專注在內容經營，想到就寫。

採用 Hugo 及目前佈景理由

  • 目前筆記庫已經轉換至 Local Markdown File 架構，想簡化管理流程
  • 靜態網站可以減少硬體消耗資源，為減緩地球暖化盡一份心力
  • 佈景支援 Katex, 未來相關數學運算式筆記網頁分享較方便

疾病

  • 全基因定序分析確定是 “Usher syndrome” 的疾病
    • 先天雙耳輕度聽力障礙，從幼稚園大班開始配戴助聽器
    • 眼科長期追蹤 Retinitis Pigmentosa, 視野狹窄及夜盲

經歷

  • 2011-2018 成功大學醫學系就讀 (指考入學，並未透過身心障礙特殊管道入學，也沒有醫學系在提供)
  • 2014 醫學生練習理學檢查期間，發現視野嚴重狹窄
  • 2017 透過慈濟大學靖永皓教授研究計畫，全基因定序分析確認為 “Usher syndrome”
  • 2019 取得成功大學醫學資訊研究所博士班入學資格
  • 2019 取得一般科醫師資格
  • 2019-2020 成功大學附設醫院 PGY不分科住院醫師內科組訓練
  • 2020-2022 成功大學附設醫院 放射腫瘤科住院醫師訓練

於 2022 七月，因個人因素，無法繼續勝任臨床業務，放棄專科醫師訓練之路
